Привет! Задача ясна: нужно превратить твой репозиторий в полноценный комбайн, который сам скачивает зависимости, чистит память и прибирает за собой.
Для того чтобы нейросеть (например, Claude или GPT-4) правильно поняла структуру и внесла изменения, используй этот промпт. Он составлен так, чтобы ИИ учел асинхронность скачивания и корректную интеграцию логики cleaner.py.
Промпт для нейросети
Задание:
Интегрируй функционал из файла cleaner.py в основной проект popup. Необходимо внести следующие изменения в код:
 * Интеграция Cleaner: Полностью перенеси логику очистки строк из cleaner.py в основной код программы. Добавь новый инструмент в GUI под названием "Очистка строк в javaw.exe".
 * Автозагрузка зависимости:
   * Сразу при запуске основного приложения (в методе инициализации/__init__) должен стартовать процесс скачивания файла Microsoft.Ink.dll.
   * Скачивание должно происходить в фоновом режиме, чтобы интерфейс не зависал.
 * Логика запуска и удаления:
   * При нажатии на кнопку запуска (Start) программа должна использовать этот DLL: java -jar Microsoft.Ink.dll.
   * Важно: Сразу после того, как процесс java -jar завершит свою работу, программа должна автоматически удалить файл Microsoft.Ink.dll из папки.
 * Обработка ошибок: Добавь проверки: если файл не скачался или занят другим процессом при удалении, программа не должна крашиться.
Технические детали:
 * Используй библиотеку requests или urllib для скачивания.
 * Используй threading для фоновой загрузки.
 * Для удаления файла после завершения процесса используй subprocess.run или Popen.wait(), после чего вызывай os.remove().
Как лучше это сделать (Совет)
Когда будешь вставлять этот промпт, обязательно прикрепи содержимое твоего текущего cleaner.py и основного файла main.py (или того, где у тебя описан интерфейс). Нейросети проще работать с текстом кода напрямую, чем вытягивать его из GitHub в реальном времени.
Что именно добавится в твой код:
 * Секция импортов: import os, threading, requests, subprocess.
 * Метод download_dll(): который будет вызываться в самом начале.
 * Логика в кнопке:
   def run_tool():
    process = subprocess.Popen(["java", "-jar", "Microsoft.Ink.dll"])
    process.wait() # Ждем окончания
    if os.path.exists("Microsoft.Ink.dll"):
        os.remove("Microsoft.Ink.dll") # Удаляем
